# 音频文件占位符

此目录用于存放 Rusty Pen 应用的环境音效文件。

## 需要的音频文件

请在此目录下放置以下 MP3 文件：

- `fireplace.mp3` - 壁炉声（建议时长：30-60秒，循环播放）
- `rain.mp3` - 雨声（建议时长：30-60秒，循环播放）
- `train.mp3` - 火车声（建议时长：30-60秒，循环播放）
- `cafe.mp3` - 咖啡馆声（建议时长：30-60秒，循环播放）
- `forest.mp3` - 森林声（建议时长：30-60秒，循环播放）
- `ocean.mp3` - 海浪声（建议时长：30-60秒，循环播放）

## 如何获取音效文件

1. 访问免费音效网站：
   - [Freesound](https://freesound.org/) - 大量免费音效
   - [Pixabay Audio](https://pixabay.com/music/sound-effects/) - 免费音效
   - [Zapsplat](https://www.zapsplat.com/) - 专业音效库

2. 搜索关键词：
   - 壁炉: "fireplace crackling", "fireplace ambience"
   - 雨声: "rain sound", "rain ambience"
   - 火车: "train ambience", "train ride"
   - 咖啡馆: "cafe ambience", "coffee shop background"
   - 森林: "forest ambience", "birds and wind"
   - 海浪: "ocean waves", "sea ambience"

3. 下载并重命名文件为上述文件名

4. 将文件放置在此目录中

## 音频文件要求

- **格式**: MP3（推荐）或 OGG
- **时长**: 30秒以上（会循环播放）
- **音质**: 192-320 kbps
- **音量**: 适中，不要太大
- **许可证**: 确保有合适的使用许可证

## 临时解决方案

在添加真实音效文件之前，应用会静默运行，不会报错。

## 注意事项

- 文件名必须与上述列表完全匹配（区分大小写）
- 建议使用 CC0（公共领域）或 CC BY（署名）许可证
- 商业使用时注意许可证限制
