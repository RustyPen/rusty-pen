# 笔具音效文件

此目录用于存放 Rusty Pen 应用的笔具打字音效文件。

## 需要的音频文件

请在此目录下放置以下 MP3 文件：

- `fountain.mp3` - 钢笔音效（建议时长：0.05-0.15秒）
- `brush.mp3` - 毛笔音效（建议时长：0.08-0.20秒）
- `feather.mp3` - 羽毛笔音效（建议时长：0.05-0.12秒）
- `ballpoint.mp3` - 圆珠笔音效（建议时长：0.04-0.10秒）

## 音效特点

### 钢笔
- 柔和的墨水流动声
- 清晰但不刺耳
- 适合长时间写作

### 毛笔
- 深沉的摩擦声
- 模拟毛笔在纸上书写
- 略带厚重的质感

### 羽毛笔
- 清脆的刮纸声
- 轻盈的笔触感
- 模拟羽毛笔的精细书写

### 圆珠笔
- 清脆的滚动声
- 简洁明快
- 模拟圆珠笔的流畅书写

## 如何获取音效文件

1. 访问免费音效网站：
   - [Freesound](https://freesound.org/) - 大量免费音效
   - [Pixabay Audio](https://pixabay.com/music/sound-effects/) - 免费音效
   - [Zapsplat](https://www.zapsplat.com/) - 专业音效库

2. 搜索关键词：
   - 钢笔: "fountain pen writing", "fountain pen click", "pen on paper"
   - 毛笔: "brush on paper", "calligraphy brush", "ink brush"
   - 羽毛笔: "quill pen", "feather pen writing", "quill on paper"
   - 圆珠笔: "ballpoint pen", "pen click", "ballpoint writing"

3. 下载并重命名文件为上述文件名

4. 将文件放置在此目录中

## 音频文件要求

- **格式**: MP3（推荐）或 OGG
- **时长**: 0.04-0.20秒（短促的打字声）
- **音质**: 128-192 kbps
- **音量**: 适中，不要太大
- **许可证**: 确保有合适的使用许可证

## 音频优化建议

1. **文件大小**: 尽量压缩，单个文件不超过 100KB
2. **采样率**: 44.1kHz 或 48kHz
3. **比特率**: 128-192 kbps
4. **格式**: 优先使用 MP3（兼容性好），备选 OGG

## 临时解决方案

在添加真实音效文件之前，应用会静默运行，不会报错。

## 注意事项

- 文件名必须与上述列表完全匹配（区分大小写）
- 建议使用 CC0（公共领域）或 CC BY（署名）许可证
- 商业使用时注意许可证限制
- 音效应该是单次打字的声音，不是连续的书写声
