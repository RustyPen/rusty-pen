## 关于

`Rusty Pen` 是一个非常有韵味的名字！它结合了两个意象：

- **Rusty（生锈的）**：暗示复古、怀旧、历经时间、略带粗粝感，也可能隐喻"久未使用"但仍有价值。
- **Pen（笔）**：代表书写、创作、思想、手稿、日记、签名、艺术。

这个名字天然适合写作、笔记、创作类工具，尤其带有极简、文艺、复古或反数字干扰的气质。

## 产品愿景

- **类型**：跨平台桌面应用（Windows、macOS、Linux）
- **风格**：极简 UI + 复古动效 + 沉浸音效
- **差异化**：把写作变成一种仪式感和收藏乐趣
- **定位**：用"老派方式"激发创作欲，模拟电报、信笺等传统书写体验
- **目标用户**：
  - 文艺青年、写作者、日记爱好者
  - 数字极简主义者（Anti-distraction）
  - 喜欢手写感但依赖数字工具的人

## 核心功能

### ✍️ 写作体验

| 功能 | 描述 |
|------|------|
| **笔具选择** | 4种笔具可选：钢笔、毛笔、羽毛笔、圆珠笔，每支笔有独特的字体样式和书写音效 |
| **纸张主题** | 4种纸张主题：复古信纸、羊皮纸、手稿纸、电报纸，每种主题有独特的背景和配色 |
| **自定义纸张背景** | 支持上传自定义背景图片，打造专属写作环境 |
| **纸张透明度调节** | 可调节纸张纹理的透明度，平衡视觉效果与可读性 |
| **A4比例纸张** | 可选择A4纸张比例，模拟真实纸张尺寸 |
| **字体系统** | 支持多种字体，包括中文字体（宋体、黑体、楷体、仿宋、微软雅黑、圆体、行书、草书、隶书、瘦金体）以及英文字体（衬线体、无衬线体、手写体等） |
| **字号选择** | 4种字号可选：小号、中号、大号、特大号 |
| **笔具音效** | 每种笔都有独特的书写音效，可开关 |

## 技术栈

- **前端框架**：React 19
- **构建工具**：Vite
- **桌面框架**：Tauri 2
- **样式**：CSS 自定义变量
- **音频**：Web Audio API 音效
- **PDF导出**：html2canvas + jsPDF
- **图标**：SVG 图标
- **国际化**：自定义I18n系统

## 快速开始

### 环境要求

- Node.js（v14 或更高版本）
- npm 或 yarn
- Rust（用于Tauri开发）

### 安装

```bash
# 克隆仓库
git clone https://github.com/yourusername/rusty-pen.git

# 进入项目目录
cd rusty-pen

# 安装依赖
npm install
```

## 开发说明

### 运行开发服务器

```bash
npm run dev
```

### 运行Tauri开发模式

```bash
npm run tauri:dev
```

### 构建桌面应用

```bash
npm run tauri:build
```

构建完成后，安装包位于 `src-tauri/target/release/bundle/` 目录。
