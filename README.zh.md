# Rusty Pen（中文版）

> 笔可能生锈，思想不会。

一个复古风格的写作应用，将传统书写的仪式感带入数字时代。

## 关于

`Rusty Pen` 是一个非常有韵味的名字！它结合了两个意象：

- **Rusty（生锈的）**：暗示复古、怀旧、历经时间、略带粗粝感，也可能隐喻"久未使用"但仍有价值。
- **Pen（笔）**：代表书写、创作、思想、手稿、日记、签名、艺术。

这个名字天然适合写作、笔记、创作类工具，尤其带有极简、文艺、复古或反数字干扰的气质。

## 产品愿景

- **类型**：跨平台写作 App（Web、Windows、macOS、iOS），可选配实体周边（如复古笔记本、金属书签、墨水色卡）
- **风格**：极简 UI + 复古动效 + 沉浸音效
- **差异化**：把写作变成一种仪式感和收藏乐趣
- **定位**：用"老派方式"激发创作欲，比如模拟打字机、电报、信笺等
- **目标用户**：
  - 文艺青年、写作者、日记爱好者
  - 数字极简主义者（Anti-distraction）
  - 喜欢手写感但依赖数字工具的人

## 核心功能

### ✍️ 写作体验设计

| 功能 | 描述 |
|------|------|
| **每日笔具解锁** | 每天解锁一种笔：钢笔、圆珠笔、羽毛笔、毛笔、铅笔、粉笔等，每支笔有独特书写质感（字体+摩擦音效） |
| **机械音效系统** | 可开关的沉浸音效：打字机回车声、羽毛笔刮纸声等 |
| **多主题** | 提供多种纸张主题（如旧版信纸、1940年代信件、1960年代手稿等），用户可自定义切换 |
| **多种纸张** | 普通纸张、带下划线的纸张、羊皮纸、复古纸张等，用户可自定义纸张样式，如纸张颜色、纸张厚度、纸张 texture |
| **自定义字体** | 支持用户上传自定义字体，应用于所有书写内容，特别是中文使用者：行书、草书、楷书、隶书、艺术字、手写体 |
| **季节性主题皮肤** | 提供秋季落叶信纸、冬日雪夜墨水等季节性主题皮肤，可供用户手动切换 |

### 📜 输出与分享

| 功能 | 描述 |
|------|------|
| **仿旧信纸导出** | 自动套用主题，生成 PDF 或图片 |
| **Rusty Pen Club 成就系统** | 连续写作 3/7/30 天，解锁虚拟古董笔、徽章，可在个人主页展示 |

## 视觉与交互设计

- **色调**：铁锈红、羊皮纸黄、墨黑、铜绿
- **字体**：衬线体为主（如 Garamond, Caslon），手写体为辅
- **动效**：缓慢淡入、纸张卷曲、墨迹晕染
- **图标**：做旧金属质感，带轻微氧化斑点
- **按钮**：提供多套样式，可搭配主题，比如圆角矩形、金属光泽、点击音效
- **输入框**：输入时，一只笔会跟随输入，模拟手写感，用户可手动调整笔的位置和角度，增加写作的真实性和趣味性
- **滚动条**：传统风格，带旧版纸张 texture，增加旧版写作的怀旧感
- **弹窗**：提供简洁、旧版风格的弹窗，增强旧版写作的怀旧感
- **启动动画**：一本书打开的动画，模拟旧版写作的仪式感
- **主界面**：提供简洁、旧版风格的主界面，包括笔具选择、纸张主题、写作区域等
- **个人主页**：用户可在个人主页展示自己的写作记录、解锁的笔具、完成的挑战等，增加旧版写作的可见性

## 快速开始

### 环境要求

- Node.js（v14 或更高版本）
- npm 或 yarn

### 安装

```bash
# 克隆仓库
git clone https://github.com/yourusername/rusty-pen.git

# 进入项目目录
cd rusty-pen

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

### 构建生产版本

```bash
# 构建项目
npm run build

# 预览生产版本
npm run preview
```
